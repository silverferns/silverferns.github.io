inout params

In-out parameters are passed as follows:


  • When the function is called, the value of the argument is copied.
  • In the body of the function, the copy is modified.
  • When the function returns, the copy’s value is assigned to the original argument.

This behaviour is known as copy-in copy-out or call by value result.

For example, when a computed property or a property with observers is passed as an in-out parameter, its


  • Getter is called as part of the function call
  • Its setter is called as part of the function return.
